## Getting Started

First, seed the database data first:

```
npm run seed
```

Second, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

run unit test
```
npm test
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
